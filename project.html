Project Overview

What is This Project?

A production-grade Retrieval Augmented Generation (RAG) system designed for enterprise knowledge management and intelligent question-answering. Built with modern technologies and best practices for scalability, security, and performance.

The system combines advanced AI capabilities with multiple retrieval strategies, multi-modal processing, and a comprehensive React-based user interface.

System Architecture

Multi-Module Design

The backend follows a clean modular architecture with clear separation of concerns:

rag-common

Shared utilities, models, and DTOs used across all modules

rag-storage

Database repositories and data access layer

rag-core

Core RAG logic, retrieval strategies, and processing pipeline

rag-api

REST API controllers, configuration, and entry point

Retrieval Strategies

Five specialized retrieval strategies with intelligent routing:

Vector Search

Semantic similarity using Weaviate

Graph Retrieval

Relationship-based using Neo4j

Hybrid (RRF)

Reciprocal Rank Fusion combining multiple sources

HyDE

Hypothetical Document Embeddings

Decomposition

Query breaking into sub-queries

AUTO

Intelligent strategy selection

Deployment

Docker Compose Setup

Complete multi-container deployment with 8 services orchestrated via Docker Compose: 

docker-compose up --build -d
Frontend (React): localhost:3000
API (Spring Boot): localhost:8080
Grafana: localhost:3001
Neo4j Browser: localhost:7474
